# 文献综述：汽车评论消费者需求挖掘

> 整理：小星 | 日期：2026-03-18

---

## 1. 研究背景

汽车评论是消费者需求的重要来源。传统方法依赖人工筛选和简单聚类，效率低且粒度粗。随着NLP技术的发展，自动化的需求挖掘成为可能。

---

## 2. 相关工作

### 2.1 情感分析 (Sentiment Analysis)

| 方法 | 文献 | 特点 |
|------|------|------|
| 传统ML | Pang+2002 | SVM/NB分类器 |
| 深度学习 | Socher+2013 | RNN系列模型 |
| 预训练模型 | Devlin+2019 | BERT系列 |
| 大模型 | ChatGPT应用 | 零样本/少样本 |

**发展趋势**：从粗粒度（整体正/负）→ 细粒度（方面级）

**主流方法**：

| 方法 | 代表文献 | 年份 |

| 基于规则 | Hu+2004 | 早期 |

## 3. 本文方法

### 3.1 整体框架

```
评论 → 价值筛选 → 属性抽取 → 需求图谱 → 决策输出
```

### 3.2 创新点

1. **价值筛选**：针对类别不平衡问题，采用多特征融合 + 阈值优化
2. **属性抽取**：结合规则词典 + LLM提示词
3. **需求图谱**：构建属性间关系网络
4. **决策支撑**：时序分析 + 对比分析

---

## 4. 数据集

- **来源**：汽车之家评论
- **规模**：6,260条
- **标注**：二分类（有价值/无价值）
- **特点**：短文本、稀疏、类别不平衡

---

## 5. 评估指标

1. **分类任务**：Precision, Recall, F1, AUC
2. **抽取任务**：准确率、召回率、F1
3. **排序任务**：NDCG
